Инициализация Slave SCCI интерфейса - установка всех параметров по умолчанию, получение дата тейбла.

Parameters
Interface- Указатель на структуру, хранящую дата тейбл, таймеры таймаута.
IOConfig- Указатель на структуру, содержащую функции обратного вызова. Эти функции передаются в интерфейс.
ServiceConfig- Указатель на структуру, хранящую параметры службы SCCI
DataTable- Указатель на дата тейбл
DataTableSize- Размер дата тейбла.
MessageTimeoutTicks- Величина таймаута при обмене сообщениями по rs232, в миллисекундах
ArgumentForCallback- Указатель на аргумент, передаваемый в функцию обратного вызова
